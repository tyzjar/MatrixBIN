#Переменные, посылаемые из весового контроллера Matrix.

Весовая система:
1) Брутто
2) Нетто
3) Тара
4) Бит стабильности
5) Статус процесса взвешивания:
   - 0 норма
   - 1 недогруз
   - 2 перегруз
6) Ошибка системы взвешивания
   - 0 нет ошибки
   - № номер ошибки
  
Система распознавания номера контейнера:
7) Номер контейнера
8) Тип распознавания:
   - 0 автоматический
   - 1 ручной ввод
9) Ошибка системы распознавания номера контейнера
   - 0  нет ошибки
   - №  номер ошибки

Система считывания номера пломбировочного устройства
10) Номер пломбировочного устройства
11) Тип ввода:
	- 0 с помощью сканера
	- 1 ручной ввод
12) Ошибка системы считывания номера пломбировочного устройства
    - 0 нет ошибки
	- № номер ошибки
	
Пункты 10-12 могут повторяться (зависит от кол-ва пломбировочных устройств)


#Переменные, нужные весовому контроллеру Matrix от контроллера Регул
1) Контейнер в ложементе
   По данному сигналу начинается процесс взвешивания контейнера:
   - совершается операция тарирования
   - начинается автоматическое определение номера контейнера
   - разрешается оператору заводить номера пломбировочных устройств
   
2) Окончание процесса взвешивания
   По данному сигналу передаюся необходимые данные в Регул.
   Сохраняются данные в БД matrix.
   
3) Аварийный останов процесса взвешивания с незавершением процесса взвешивания
	По данному сигналу все данные по нчатому процессу взвешивания сбрасываются.
