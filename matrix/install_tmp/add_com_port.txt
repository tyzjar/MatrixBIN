Отсутствую файлы дополнительных портов ttyS от PCI контроллера
Спасибо, все заработало! Действительно просто ограничение на количество создаваемых портов. Напиши полную инструкцию, вдруг кому пригодится. Для того чтобы изменить параметры загрузки ядра, необходимо внести изменение в файл загрузчика grub:

sudo nano /etc/default/grub
Внутри файла добавить строку с указание необходимо числа портов, в моем случай 8:

GRUB_CMDLINE_LINUX_DEFAULT="8250.nr_uarts=8"
Далее необходимо обязательно обновить загрузчик:

sudo update-grub
После чего перезагружаем систему и готово:

sudo reboot
